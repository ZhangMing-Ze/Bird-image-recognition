# 🐦 鸟类分类模型训练说明（PyTorch + ResNet18）

本项目基于 **PyTorch + ResNet18** 实现一个 **165 类鸟类图像分类模型**，支持训练、验证、日志记录和模型保存，并兼容 GPU / CPU 环境。

---

## 📌 一、项目功能

* **数据集处理**：加载自定义图像数据集，自动划分 80% 训练集、20% 验证集
* **模型架构**：基于预训练 ResNet18 微调，适配 165 分类任务
* **训练优化**：

  * 分层学习率（骨干网络 + 分类头）
  * Adam 优化器
  * 交叉熵损失函数
* **可视化**：TensorBoard 实时记录损失 / 准确率曲线
* **进度展示**：tqdm 训练进度条
* **模型保存**：自动保存最优模型权重

---

## ⚙️ 二、环境依赖

安装以下 Python 库：

```bash
pip install torch torchvision matplotlib tqdm tensorboard
```

## 📂 三、数据集准备

数据集目录结构（必须严格遵循）：

```
D:\homework\          # 数据根目录（可修改 data_dir）
└── train\
    ├── 鸟类类别1\
    │   ├── img1.jpg
    │   └── img2.jpg
    ├── 鸟类类别2\
    └── ...
```

> ⚠️ 所有类别图像必须放在 `train` 文件夹下，程序会自动划分训练集 / 验证集

## 🧠 四、核心配置参数

```python

# 训练参数
batch_size = 64
epochs = 50
learning_rate = 1e-5

# 模型保存路径

```
